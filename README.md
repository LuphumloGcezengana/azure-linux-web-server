# Azure Linux Web Server Project

## Overview
In this project, I deployed a Linux-based web server on Microsoft Azure using a virtual machine and Apache. This project helped me gain hands-on experience with cloud infrastructure and web hosting.

---

## Technologies Used
- Microsoft Azure  
- Ubuntu Linux  
- Apache Web Server  
- SSH  
- Networking (NSG, port configuration)

---

## ⚙️ What I Did
- Created an Azure Virtual Machine  
- Connected to the VM using SSH  
- Installed and configured Apache  
- Opened port 80 for web traffic  
- Deployed a custom HTML website  

---

## ⚠️ Challenges & Solutions
Apache initially failed to start due to a port conflict.

- Issue: Port 80 was already in use  
- Cause: NGINX was running  
- Solution: Stopped and disabled NGINX, then restarted Apache  

This gave me real-world troubleshooting experience.

---

## Result
Successfully hosted a live website on Azure using a public IP address.
